# 📊 Nigeria FX Market (NFEM) Analysis

An end-to-end data analysis project exploring the Nigeria Financial Exchange Market. This project transforms raw CBN data into actionable financial insights regarding exchange rate volatility and market liquidity.

## 🚀 Key Features
- **Data Pipeline:** Automated cleaning and transformation of NFEM turnover and rate data.
- **Local AI Analysis:** Integration with **Ollama** running the **Qwen** model locally to generate financial narratives directly from the data.
- **Volatility Analysis:** Statistical tracking of exchange rate fluctuations.
- **Automated Reporting:** Python-driven PDF generation for executive summaries.

## 🛠️ Tech Stack
- **Language:** Python (Pandas, NumPy)
- **Local LLM:** Ollama (Model: Qwen)
- **Reporting:** ReportLab / WeasyPrint
- **Environment:** Jupyter Notebook

## 📈 Local AI Integration
Unlike projects using cloud APIs, this project utilizes a local AI stack. By running **Qwen via Ollama**, the analysis remains private, secure, and cost-effective, demonstrating a "Private-AI" approach to financial data science.
